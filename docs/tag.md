# 4. Tag Your Release on GitHub

1. Create a Git tag:

    - A Git tag is required for JitPack to identify the version to publish.

    ```bash
    git tag v1.0.0
    git push origin v1.0.0
    ```

2. Verify the tag on GitHub:
